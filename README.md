# Iris Species Classification
Proyecto de Machine Learning desarrollado con Streamlit y Random Forest.

## Descripcion

Este proyecto clasifica especies de flores Iris utilizando un modelo de Machine Learning Basado en las siguientes caracteristicas:

 - Longitud del sepalo
 - Ancho del sepalo
 - Longitud del petalo
 - Ancho del petalo

 La aplicacion permite al usuario ingresar medidas de una flor y obtener una prediccion de la especie correspondiente.

 ## Tecnologias Utilizadas

 - Python
 - Streamlit
 - Pandas
 - Scitkit-Learn
 - Plotly

 ## Funcionalidades

 - Dashboard interactivo
 - Modelo de clasificacion Random Forest
 - Metricas:
   - Accuracy
   - Precision
   - Recall
   - F1 Score
- Grafico 3D interactivo
- Histograma de distribucion
